### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/Rashmina2000/Expense_tracker_flutter-app.git
   ```
2. Navigate into the directory:
   ```sh
   cd Expense_tracker_flutter-app
   ```
3. Get the dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app:
   ```sh
   flutter run
   ```
